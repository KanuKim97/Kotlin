## Design Pattern
  ### MVVM Pattern 
    - Model + View + ViewModel 
    - Model : Application의 데이터와 스 데이터를 처리하는 부분 
    - View : 사용자에게 보여지는 UI 부분
    - ViewModel : View를 표현하기 위해 만든 View를 위한 Model
